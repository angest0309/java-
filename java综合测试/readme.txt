题目介绍：
编写一个 Java 应用程序，实现对某个目录中的所有 Java 源程序文件
（包含该目录的子目录中的源程序文件）进行统计。
具体说明：本实验要求对目录中 Java 源程序文件进行统计，统计内容
包括：(1) 目录中每个源程序 文件的总行数和空白行数，文件的字节
数；(2) 目录中所有源程序文件合计总行数、合计空 白行数、合计文
件的字节数。同时还需要将统计分析的结果保存在根目录的result文件
夹中， 方便后续的读取。

其中CodeAnalyzer是可以分析各个java文件的大小，代码行数、空白
行数、注释行数等，并在每个java文件统计完输出结果，且会将所有
java文件一起统计总记的文件数目、代码行数、空白代码行数、字节数。
会自动将输出结果的两个txt文件创建在result文件里。
并会将输出结果传输到文件result.txt和result1.txt。
